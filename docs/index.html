IoU & Confusion Matrix
// Intersection over Union
IoU = |A ∩ B| / |A ∪ B|
= Intersection Area / Union Area
─ IoU ≥ 0.5 → TP 판정 (일반 기준)
─ IoU < 0.5 → FP 판정
─ 미탐지 GT → FN 판정
TP: IoU ≥ θ FP: IoU < θ FN: 미탐지

03
의료 영상 분할 Loss Function
// 클래스 불균형 문제 (의료 영상)
배경(Background)이 압도적으로 넓고, 관심 객체(Cell/Tumor)는 극히 작은 비율 차지
Background (배경) — 95%+
Object — 5%↓
→ CE loss만 쓰면 모델이 배경만 예측해도 loss가 낮음 → Dice Loss 병행 필요
Cross Entropy
L_CE = −[y·log(p̂) + (1−y)·log(1−p̂)]

y : ground truth (0 or 1)
p̂ : 예측 확률
픽셀 단위 분류 손실.
각 픽셀을 독립적으로 평가하기 때문에 배경 픽셀이 많을수록 배경에 치우쳐 학습됨.
픽셀 독립 평가 불균형에 취약
Dice Loss
Dice = 2|A∩B| / (|A|+|B|)
L_Dice = 1 − Dice

A : 예측 마스크 (pred)
B : 실제 마스크 (GT)
IoU와 유사하게 겹침(overlap)을 직접 최적화.
클래스 빈도와 무관하게 작은 객체도 동등한 가중치로 학습 가능.
Overlap 최적화 불균형 강건
// 실전: 결합 손실 함수 (Combined Loss)
L_Total
최종 손실
=
α · L_CE

// COCO vs Kaggle 평가 방식 비교
COCO AP@[0.5:0.05:0.95] — 10개 임계값 평균
0.50 0.55 0.60 0.65 0.70 0.75 0.80 0.85 0.90 0.95
mAP_COCO = (1/10) Σ AP@θᵢ
Kaggle Sartorius — 6개 임계값 평균
0.50 0.55 0.60 0.65 0.70 0.75
mAP_Sartorius = (1/6) Σ AP@θᵢ
AP = ∫₀¹ p(r) dr ≈ Σ (rₙ − rₙ₋₁)·pₙ
─ 11-point interpolation 또는 all-point interpolation
─ mAP = mean AP over all classes
// AP 시각화 (PR Curve AUC)
AP@0.5: 느슨한 기준 → 박스가 대략 맞으면 TP (일반 탐지)
AP@0.75: 엄격한 기준 → 박스가 정교하게 맞아야 TP (정밀 분할)
AP@0.5:0.95: 균형적 평가 → COCO 표준
